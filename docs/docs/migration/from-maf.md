# Migrating from Other .NET Agent Frameworks

Migration guides for developers coming from other .NET agent frameworks are coming soon.

If you're evaluating Strands Agents .NET from another framework, the [Getting Started](../getting-started) guide and [Concepts](../concepts/agent-event-loop) section are the fastest way to get oriented. The core ideas — agent, tool, event loop, session — map closely to what you already know.

Questions? Start a thread in [GitHub Discussions](https://github.com/apncodes/StrandsAgents.net/discussions).
